#项目发布平台
---

##简介

使用CodeIgniter和MySQL搭建的简单项目发布平台。数据库定义见projects.sql，导出的测试数据在data.sql（数据库名：project_issue）。

前台界面`<base_url()>`	后台登录`<base_url()>/index.php/admin/index`
